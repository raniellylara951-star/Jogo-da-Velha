# Jogo da Velha em React

**Descrição**

Projeto desenvolvido em React como atividade prática da disciplina, implementando um jogo da velha com funcionalidades extras para melhorar a experiência do usuário e demonstrar o uso de componentes, estados e eventos em React.

**Funcionalidades**

- Reiniciar jogo;
- Exibir jogador da vez;
- Bloquear jogadas após vitória ou empate;
- Verificação completa das condições de vitória;
- Linhas;
- Colunas;
- Diagonais;
- Placar de partidas;
- Vitórias do jogador X;
- Vitórias do jogador O;
- empates;
- Destaque visual para a combinação vencedora;
- Histórico de jogadas;
- Desfazer última jogada;
- Modo de jogo contra a máquina.

**Tecnologias Utilizadas**

- React;
- JavaScript;
- CSS;
- HTML.

**Objetivo**

Aplicar conceitos de React, componentização, gerenciamento de estado e manipulação de eventos por meio do desenvolvimento de um jogo da velha interativo.
